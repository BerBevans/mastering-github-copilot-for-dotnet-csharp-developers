通过 GitHub Copilot for Azure 学习云端部署——这是您实现云端成功的终极指南。从建立强大的基础开始，提高编码效率，然后过渡到轻松的应用程序部署，利用 Azure 的强大可扩展性。最后，学习高级故障排除技术，确保您的应用程序可靠且性能优异。有了 Copilot 作为您的可靠助手，您将自信地创建、部署和管理一流的应用程序，在每个阶段都提升您的云端技能。

- **适合人群**：开发者和运维人员（ITPRO）
- **您将学到的内容**：通过三个关键阶段——学习、部署和故障排除，全面掌握 GitHub Copilot for Azure。
- **您将构建的内容**：自信地构建、部署和维护高质量应用程序。

## 学习目标

在本次工作坊中，您将：

- 掌握 GitHub Copilot for Azure 使用的基础概念和最佳实践。

- 使用高级工具诊断并解决问题，确保可靠的性能。

<a name="pre-req"/>

## 先决条件

- [GitHub Copilot Chat](https://marketplace.visualstudio.com/items?itemName=GitHub.copilot)：GitHub Copilot 是一个 AI 编程助手工具，可以帮助您更快更智能地编写代码。
- [.NET 安装工具](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.vscode-dotnet-runtime)：此扩展为其他扩展（如 C# 和 C# Dev Kit 扩展）提供了一种统一的方式来安装本地版本的 .NET 运行时以及全局版本的 .NET SDK。这些扩展会告诉 .NET 安装工具它们需要的 .NET SDK，如果设备上没有匹配的版本，我们会为您安装。用户也可以通过阅读下方内容自行安装 .NET SDK。
- [GitHub Copilot for Azure](https://marketplace.visualstudio.com/items?itemName=ms-azuretools.vscode-azure-github-copilot)：GitHub Copilot for Azure 是 @azure 扩展，旨在帮助简化 Azure 开发的流程。您可以直接在 Visual Studio Code 中向 @azure 提问有关 Azure 服务的问题，或获取与 Azure 开发相关任务的帮助。
- [Microsoft Azure 账户](https://azure.microsoft.com/pricing/purchase-options/azure-account)：通过 Azure 的可扩展且高效的服务，无缝地构建、部署和管理应用程序。免费试用 Azure 最多 30 天，无需预付款，可随时取消。
